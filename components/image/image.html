Image

Responsive Image

Add responsive-image class to the img tag. The image will grow to 100% width of it's parent container. 

            
    <div class="...">
        <img class="responsive-image" src="..." alt="..." />
    </div>

Round Image

Add round-image class to the img tag to get a round image which will grow to 100% width of it's parent container.

            
    <div class="...">
        <img class="round-image" src="..." alt="..." />
    </div>
